# Spotilol

A lightweight Spotify Web wrapper for Windows with ad-blocking and unlimited skips.

> This project is closed source to prevent detection and patching by Spotify's security measures.

## Features

- Ad-free listening experience
- Unlimited skips
- Track change notifications
- System tray integration
- Global media key shortcuts
- Minimal resource usage

## Download

Get the latest version from the [Releases](https://github.com/lyssadev/Spotilol/releases) page.

Available formats:
- Installer (.exe)
- Portable (.exe)

## Requirements

- Windows 10/11 (64-bit)
- Internet connection
- Spotify account (free or premium - Optional for premium lol)
